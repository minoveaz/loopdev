feat: Add certified ModuleWorkspace component

Adds the new `ModuleWorkspace` layout component to the UI library. This industrial-grade shell provides a robust and accessible foundation for building module interfaces, featuring:

- Deterministic Push (desktop) and Overlay (mobile) modes.
- Built-in accessible Focus Trap using Radix UI.
- Centralized "Topmost" state management for panels.
- Full tokenization for easy theming and customization.
