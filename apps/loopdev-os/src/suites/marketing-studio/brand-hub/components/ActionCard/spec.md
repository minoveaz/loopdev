# Spec: ActionCard (v1.0)

## 1. Purpose
High-impact action trigger for primary workflows (e.g., Create Draft, Compare).

## 2. Visual Rules
- **Icon:** Large (20px) centered in a boxed container.
- **Intent Primary:** High contrast border and shadow.
- **Micro-interaction:** Card lifts sutilly on hover.

## 3. Hierarchy
1. Title (Sm, bold)
2. Description (Xs, muted)
