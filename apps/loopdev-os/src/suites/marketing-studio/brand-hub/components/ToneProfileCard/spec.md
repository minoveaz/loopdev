# Spec: ToneProfileCard (v1.0)

## 1. Purpose
Operational guidance for brand voice. Shows how to translate narrative into specific linguistic behavior.

## 2. Visual Rules
- **Structure:** Header (Name + Desc) + Split Section (Do / Don't).
- **Do Section:** Uses `text-emerald-500` or green check icon indicator.
- **Don't Section:** Uses `text-red-500` or red X icon indicator.
- **Micro-interaction:** Card highlights on hover.

## 3. Interaction
- Clicking the card opens the **Inspector** on the **Impact** tab (Consumers of this tone).
